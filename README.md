# 🔥 Horny Diffusion - Generador NSFW de Imágenes

Generador local de imágenes porno con Stable Diffusion.

**Características:**
- Sin censura (safety checker desactivado)
- Soporte para LoRAs y embeddings
- Interfaz oscura y cachonda
- Muy fácil de usar

## Instalación

```bash
git clone https://github.com/tuusuario/nsfw-ai-image-generator.git
cd nsfw-ai-image-generator
pip install -r requirements.txt
python app.py
